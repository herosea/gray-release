#### 项目规范

添加数据库规范到记忆文件 AGETNT.md

```
- 数据库中表必须包含：
  - `id`（自增无符号主键）
  - `created_time` timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间'
  - `updated_time` timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间'
- 所有表名必须以 `bkk_` 开头（示例：`bkk_chat_memory`, `bkk_gray_invite_code`）
- 用户的id用在数据库中用 uid 类型 bigint UNSIGNED
  
将这个数据库建表规范添加到 AGETNT.md 中
```

*H2 数据库不支持 BIGINT UNSIGNED 语法
