# AI 灰测后台接口设计

> 创建时间：2025-12-24
> 更新时间：2026-03-02

## 1. 模块说明

后台邀请码管理接口统一前缀：`/api/admin/invite-codes`

## 2. 接口清单

### 2.1 新建邀请码
- Method: `POST`
- Path: `/api/admin/invite-codes`
- Body:
```json
{
  "channel": "XHS",
  "code": "XHS888",
  "totalQuota": 100,
  "alertThreshold": 10,
  "operator": "admin"
}
```
- 规则:
  - `code` 为空时自动随机生成
  - `code` 不为空时按手动码处理，必须唯一

### 2.2 邀请码分页列表
- Method: `GET`
- Path: `/api/admin/invite-codes`
- Query:
  - `keyword`(可选): 按邀请码/渠道模糊匹配
  - `channel`(可选)
  - `status`(可选): `ENABLED`/`DISABLED`
  - `page`(默认1)
  - `size`(默认10)

### 2.3 邀请码详情
- Method: `GET`
- Path: `/api/admin/invite-codes/{id}`

### 2.4 编辑邀请码
- Method: `PUT`
- Path: `/api/admin/invite-codes/{id}`
- Body:
```json
{
  "channel": "XHS",
  "code": "XHS888",
  "totalQuota": 120,
  "alertThreshold": 10,
  "operator": "admin"
}
```
- 规则:
  - `usedQuota > 0` 时禁止修改 `channel` 与 `code`
  - 始终允许调整 `totalQuota` 与 `alertThreshold`

### 2.5 启用/禁用
- Method: `PATCH`
- Path: `/api/admin/invite-codes/{id}/status`
- Body:
```json
{
  "status": "DISABLED",
  "operator": "admin"
}
```

### 2.6 删除邀请码
- Method: `DELETE`
- Path: `/api/admin/invite-codes/{id}`
- 规则:
  - 仅 `usedQuota == 0` 允许删除
  - 逻辑删除（`deleted = 1`）

### 2.7 随机生成邀请码
- Method: `POST`
- Path: `/api/admin/invite-codes/generate`
- Body:
```json
{
  "count": 5
}
```
- 返回：随机码数组（仅生成，不落库）

## 3. 错误码

- `INVITE_CODE_DUPLICATE`: 邀请码重复
- `INVITE_CODE_IMMUTABLE_FIELD`: 已使用邀请码的不可变字段被修改
- `INVITE_CODE_QUOTA_INVALID`: 额度非法
- `INVITE_CODE_ALERT_THRESHOLD_INVALID`: 告警阈值非法
- `INVITE_CODE_DELETE_FORBIDDEN`: 已使用邀请码不可删除
- `INVITE_CODE_NOT_FOUND`: 邀请码不存在
- `INTERNAL_ERROR`: 服务端兜底异常
